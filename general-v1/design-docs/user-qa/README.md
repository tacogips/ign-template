# User Q&A

This directory contains items requiring user confirmation or decision.

## Purpose

Store questions, pending decisions, and items awaiting user approval.

## File Naming Convention

| Prefix | Use Case |
|--------|----------|
| `qa-` | Questions or confirmation items |
| `pending-` | Pending decisions |

## Current Items

- [qa-example.md](./qa-example.md) - Example: Source Access Scope
- [pending-example.md](./pending-example.md) - Example: Evidence Capture Format

## Adding New Items

1. Create a new file with the appropriate prefix
2. Include a clear description of the question or decision needed
3. List available options if applicable
4. Update this README with the new item
